## Быстрый старт (1 кворк)

1. Подкатегория: **Telegram Mini Apps**
2. Открой `telegram-mini-apps.txt`
3. Копируй название → описание → цену → FAQ
4. Загрузи обложку (`cover-mini-app.png` или `cover-telegram-boty.png`) + галерею
5. Опубликуй

## Несколько карточек

Kwork разрешает несколько кворков в одной рубрике — можно выложить Mini Apps + Чат-боты + Скрипты. Не дублируй одинаковые тексты: в каждом файле свой акцент.
